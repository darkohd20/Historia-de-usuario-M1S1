# User-history-M1S1
This program was created to practice user story M1S1, validating user input for a product name, price, and quantity, then calculating and displaying the total cost.
What does it do?
The program asks the user to enter a product name, a unit price, and a quantity. It validates that:

The name is not empty
The price is a valid decimal number
The quantity is a valid whole number

Once all values are entered correctly, it calculates the total cost (price × quantity) and displays a summary with all the information.
How is it used?

1.Run the program in your terminal with python inventory.py
2.Enter the product name when prompted
3.Enter the unit price (example: 10.00)
4.Enter the quantity (example: 5)
5.The program will display the product name, unit price, quantity, and total cost


If you enter an invalid value, the program will show an error message and ask you to try again.
